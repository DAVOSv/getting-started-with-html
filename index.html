How the internet works

The internet is a group of computers that work with each other to transfer information. The world wide web uses HTML to transfer information back and forth. First a computer will link to a web page, that sends an http request to a server ( a computer that hosts the requested information.) The server sends the information back to the computer. The information is interpreted through a web browser and displays the information in a visual format.

HTML Markup

A tag is a reference to a purpose made in an html document that tells the website what to do with the following information.

An element is a line of code with both an opening and closing tag and some content for the tags to act on.



Specificity

Computers need exact instructions to operate as intended. This applies to the internet as well. One wrong direction or typo can have major unintended consequences.

HTML Attributes

Each HTML element is either inline or block. Inline is just for text. Block creatles a box around the element and can work with both text and other kinds of content
